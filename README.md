# 🧠 RAG система (Retrieval-Augmented Generation)

## 🚀 AI-система для поиска и генерации ответов

---

## 📖 Описание
Проект реализует систему **Retrieval-Augmented Generation (RAG)** для ответа на вопросы по базе документов.

Система сочетает:
- 🔍 семантический поиск (retrieval)
- 🤖 генерацию ответа (LLM)

👉 Это позволяет получать **точные и контекстные ответы**, снижая hallucinations.

---

## ⚙️ Архитектура
```
Запрос пользователя
        ↓
Эмбеддинг запроса
        ↓
Векторный поиск
        ↓
Релевантные документы
        ↓
LLM (с контекстом)
        ↓
Ответ
```

---

## 🔧 Функционал

- 📄 Разбиение документов на чанки  
- 🧠 Построение эмбеддингов  
- 🔍 Семантический поиск  
- 📦 Извлечение контекста  
- 🤖 Генерация ответа через LLM  
- 🎯 Prompt engineering
